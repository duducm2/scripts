## WhatsApp Desktop

**Modifier:** Shift

These shortcuts work when WhatsApp Desktop (Chrome-based) is the active window.

### Shift + V — Toggle Voice Message Recording

Starts recording a voice message if not currently recording. If already recording, stops the recording and sends the voice message. This provides a quick way to record and send voice messages without needing to hold down any button.

### Shift + S — Search Chats

Opens the chat search interface, allowing you to quickly find conversations by typing keywords. Focuses the search field so you can immediately start typing your search query.

### Shift + R — Reply to Message

Activates the reply function for the currently selected message. This opens the reply composer with the original message quoted, ready for you to type your response.

### Shift + E — Open Emoji Panel

Opens the emoji selector panel, allowing you to browse and insert emojis into your message. Provides quick access to the full emoji library without using the mouse.

### Shift + U — Toggle Unread Filter

Toggles between showing all conversations and showing only unread conversations. When the unread filter is active, only chats with unread messages are displayed in the conversation list.

### Shift + F — Focus Current Chat

Moves the keyboard focus to the current conversation in the chat list. Useful for keyboard navigation when you need to quickly access chat-related actions.

### Shift + M — Mark as Read/Unread

Toggles the read status of the current conversation. If the chat is marked as read, it becomes unread; if it's unread, it becomes read.

### Shift + P — Pin/Unpin Chat

Pins the current conversation to the top of your chat list, or unpins it if already pinned. Pinned chats remain at the top for quick access.

## Microsoft Outlook

Outlook shortcuts are context-aware and change behavior depending on which Outlook window is active (main window, reminder, appointment, or message).

### Main Window (Shift)

#### Shift + G — Send to General Folder

Quickly moves the selected email to your "00 General" folder. This is a rapid filing shortcut that navigates the folder structure and confirms the move automatically.

#### Shift + N — Send to Newsletter Folder

Moves the selected email to your "01 Newsletter" folder. Useful for organizing subscription emails and newsletters into a dedicated location.

#### Shift + I — Go to Inbox

Navigates directly to your Inbox folder from anywhere in Outlook. Clears any current navigation and focuses on your main inbox view.

#### Shift + S — Focus Subject Field

Places the cursor in the Subject/Title field of an email or message. Allows you to quickly edit or view the subject line.

#### Shift + T — Focus To/Required Field

Moves focus to the recipient field (To or Required attendees). Useful when composing emails or meeting invitations.

#### Shift + B — Focus Body Field

Navigates from the Subject field to the message body. First focuses on Subject, then tabs to the body area for immediate typing.

#### Shift + F — Toggle Focused/Other

Switches between the "Focused" and "Other" inbox views. Outlook's Focused Inbox separates important emails from less important ones.

#### Shift + D — Don't Send Response

For meeting invitations, quickly selects the option to not send any response to the organizer. Shows a brief loading indicator during the action.

#### Shift + E — Send Response

For meeting invitations, sends your response (accept/decline) to the meeting organizer. Shows a brief loading indicator while processing.

#### Shift + K — Cycle Panes Backward

Navigates backward through Outlook's pane layout (navigation, reading pane, folder list, etc.).

#### Shift + L — Cycle Panes Forward

Navigates forward through Outlook's different panes and interface sections.

### Reminder Window (Shift)

These shortcuts work when an Outlook reminder popup is active.

#### Shift + H — Snooze 1 Hour

Snoozes the current reminder for exactly one hour. Shows a confirmation dialog before applying.

#### Shift + F — Snooze Four Hours

Snoozes the current reminder for four hours. Useful for deferring items until later in the day.

#### Shift + D — Snooze 1 Day

Snoozes the reminder until tomorrow at the same time. Shows confirmation before applying.

#### Shift + X — Dismiss All Reminders

Dismisses all currently showing reminders at once. Requires confirmation to prevent accidental dismissal.

### Appointment/Meeting Window (Shift)

These shortcuts work when editing an Outlook appointment, meeting, or event.

#### Shift + S — Focus Start Date

Clicks on the start date combo box, allowing you to modify when the appointment begins.

#### Shift + P — Open Start Date Picker

Opens the calendar date picker for the start date, allowing visual date selection.

#### Shift + T — Focus Start Time

Focuses the start time combo box to modify the appointment's start time.

#### Shift + E — Focus End Date

Focuses the end date field to modify when the appointment ends.

#### Shift + H — Focus End Time (Hour)

Focuses the end time combo box to change the ending hour of the appointment.

#### Shift + A — Toggle All-Day Event

Toggles the "All day" checkbox. When checked, the appointment spans the entire day without specific times.

#### Shift + I — Focus Title Field

Places the cursor in the appointment title/subject field for editing.

#### Shift + R — Focus Required/To Field

Focuses the attendee field where you can add meeting participants.

#### Shift + L — Focus Location, Then Body

First focuses the Location field, then tabs to the body area for entering meeting details.

#### Shift + B — Focus Body (from Location)

Navigates from the Location field to the meeting body/notes area.

#### Shift + C — Make Recurring

Opens the recurrence dialog to convert a one-time appointment into a recurring event.

#### Shift + W — Appointment Configuration Wizard

Opens a multi-step wizard that guides you through configuring appointment settings: Status (Free/Busy/Out of Office), Privacy (Private on/off), All-day toggle, Category, Reminder time, and Note marker.

### Message Window (Shift)

These shortcuts work when composing or reading an email message.

#### Shift + S — Focus Subject Field

Places the cursor in the email subject line for editing.

#### Shift + T — Focus To Field

Focuses the recipient field to add or modify email recipients.

#### Shift + B — Focus Body

Navigates from the Subject field to the message body for composing content.

## Microsoft Teams

Teams shortcuts are context-aware, working differently in meeting windows versus chat windows.

### Meeting Window (Shift)

#### Shift + C — Open Chat Pane

Opens the in-meeting chat panel, allowing you to send and read messages during the meeting.

#### Shift + M — Maximize Meeting Window

When viewing a compacted/minimized meeting window, switches to the full meeting window. Useful when Teams shows the small overlay view.

#### Shift + R — Open React Menu

Opens the reactions menu to send emoji reactions during a meeting (thumbs up, heart, applause, etc.).

#### Shift + J — Join Now (Camera & Mic On)

Clicks the "Join now" button in the pre-join lobby with camera and microphone enabled. Quickly joins a meeting without adjusting settings.

#### Shift + A — Audio Settings

Opens the audio settings menu in the pre-join screen, allowing you to select microphone and speaker devices before joining.

### Chat Window (Shift)

#### Shift + R — Reply to Message

Opens the reply composer for the current message context. Navigates through the message and initiates a reply.

#### Shift + U — View All Unread Items

Filters the chat list to show only conversations with unread messages.

#### Shift + P — Pin Chat

Pins the current chat to the top of your chat list. Opens the context menu and selects the pin option.

#### Shift + E — Edit Message

Opens the edit interface for a recently sent message, allowing you to modify its content.

#### Shift + A — Attach File

Opens the file attachment interface to share files in the current conversation.

#### Shift + H — Open History Menu

Opens the navigation history menu, showing recently visited chats and channels.

#### Shift + M — Mark as Unread

Marks the current conversation as unread so it appears highlighted in your chat list.

#### Shift + X — Unpin Chat

Removes the pin from a previously pinned chat, returning it to normal sort order.

#### Shift + C — Collapse All Conversation Folders

Collapses all expanded folders and categories in the chat navigation panel, providing a cleaner view.

#### Shift + I — Toggle Info/Details Panel

Opens or closes the conversation details panel, showing participant information and shared content.

#### Shift + . — Detach Chat to New Window

Opens the current conversation in a separate, standalone window. Useful for multi-tasking.

#### Shift + T — Add Participants

Opens the dialog to view current participants and add new people to the conversation.

#### Shift + V — Video Call

Initiates a video call with the current chat participant(s). Shows a confirmation dialog first.

#### Shift + F — Fold Chat Sections

Collapses all expandable sections in the chat navigation (like Pinned, Recent, etc.) and navigates to the home position.

#### Shift + O — Open Home Panel

Navigates to the Teams home panel and moves focus to the top of the chat list.

#### Shift + L — Like Reaction

Quickly sends a "Like" (thumbs up) reaction to the current message.

#### Shift + G — Heart Reaction

Sends a heart emoji reaction to the current message.

#### Shift + J — Laugh Reaction

Sends a laughing emoji reaction to the current message.

## OneNote

### Shift + P — Select Line and Children

Selects the current paragraph/line and all its child (indented) content. Useful for selecting entire hierarchical note structures.

Opens the search dialog, clears any existing search, and automatically inserts double quotes for exact phrase matching. Places the cursor between the quotes for immediate typing.

Selects the current line and all its indented children, then deletes them. A quick way to remove entire note sections.

Deletes just the current line while preserving all child (indented) content. Children move up to take the deleted line's position.

Collapses the current outline section, hiding all child content under a collapsed header.

Expands the current collapsed section, revealing all hidden child content.

### Shift + I — Collapse All Sections

Collapses all outline sections on the current page, showing only top-level headers.

### Shift + O — Expand All Sections

Expands all collapsed sections on the current page, revealing all content at every level.

## Cursor (Code Editor)

Cursor shortcuts use various modifiers (Shift, Ctrl, Ctrl+Alt) for different functions.

### Ctrl Shortcuts (Ctrl key)

#### Ctrl + 1 — Focus on Code

Removes UI clutter by closing panels, hiding notifications, and focusing attention on the code editor. Sends multiple cleanup commands.

#### Ctrl + 5 — Reload

Clicks the Reload button to refresh the current view or reload extensions.

#### Ctrl + , — Fold All Directories

Collapses all directory trees in the File Explorer panel, providing a clean, compact view of the file structure.

#### Ctrl + Q — Unfold All Directories

Expands all collapsed directories in the File Explorer, showing the complete file tree.

#### Ctrl + 6 — Agent Panel Actions

Context-aware agent panel interaction. If the toggle is visible, opens the editor; otherwise, opens the agent panel and maximizes chat size.

#### Ctrl + M — Git Commit Workflow

Comprehensive commit workflow: Opens Git section, generates commit message using AI, waits for message generation, then prompts whether to push after committing. Blocks input during automation to prevent interference.

### Shift Shortcuts

#### Shift + F — Fold Code Block

Collapses the current code block or function, hiding its contents under a folded indicator.

#### Shift + U — Unfold Code Block

Expands a folded code block, revealing its hidden contents.

#### Shift + M — Markdown Preview

Opens a side-by-side markdown preview of the current markdown file.

#### Shift + W — Move Editor to New Window

Detaches the current editor tab and opens it in a new Cursor window.

#### Shift + T — Focus Terminal

Switches focus to the integrated terminal panel. Opens the terminal if not visible.

#### Shift + N — New Terminal

Creates a new terminal instance in the integrated terminal panel.

#### Shift + E — Focus File Explorer

Switches focus to the File Explorer sidebar, allowing keyboard navigation of files.

#### Shift + K — Markdown Preview + New Window

Opens markdown preview, waits for it to load, then moves the preview to a maximized new window.

#### Shift + C — Command Palette

Opens Cursor's command palette for accessing all commands and features.

#### Shift + X — Expand Selection

Incrementally expands the current text selection to include larger syntax structures.

#### Shift + S — Go to Symbol

Opens the symbol search in the current file for quick navigation to functions, classes, etc.

#### Shift + H — Show Chat History

Opens the AI chat history, showing previous conversations with the assistant.

#### Shift + I — Paste Image

Activates the image paste functionality for inserting images into the editor.

#### Shift + G — Fold Git Directories (SCM)

Collapses all Git repository sections in the Source Control view.

#### Shift + Q — Search in Files

Opens the global search panel for searching across all files in the workspace.

#### Shift + R — Breadcrumbs Menu

Opens the breadcrumbs navigation menu showing the current file's location in the project hierarchy.

#### Shift + D — Git Section

Opens or focuses the Git/Source Control section.

#### Shift + Z — Close All Editors

Closes all open editor tabs at once.

#### Shift + Y — Zen Mode

Enters Zen mode, hiding all UI elements except the editor for distraction-free coding.

#### Shift + P — Git Pull

Executes a Git pull operation to fetch and merge remote changes.

#### Shift + V — Git Commit

Opens the Git commit interface to stage and commit changes.

#### Shift + B — Git Push

Pushes committed changes to the remote repository.

#### Shift + O — Emoji Selector

Opens a quick emoji picker with common task-related emojis (checkboxes, timers, priorities, questions) for inserting into code comments or documentation.

#### Shift + A — Switch AI Models

Opens the AI model selector to switch between different language models (Claude, GPT, etc.).

#### Alt + N — Review Next File

Navigates to the next file in a code review or diff view.
